Ты — эксперт по разработке на Node.js и React. Мне нужно добавить в мой мессенджер на Replit функционал «Подарки», как в Telegram.

Стек моего мессенджера (выбери нужное или укажи свой):
- Бэкенд: Node.js + Express.js (или я использую Next.js API routes)
- База данных: PostgreSQL через Drizzle ORM (или Prisma / Supabase)
- Фронтенд: React (Web) или React Native (если мобильное приложение)
- Аутентификация: уже есть (сессии или JWT)
- Внутренняя валюта: «звёзды» (у каждого пользователя поле stars)

Задача: создать полноценную систему подарков, включая:

1. Панель администратора (или API-эндпоинты) для добавления подарков:
   - название подарка
   - цена в звездах
   - ссылка на Lottie-анимацию (JSON) или видео-превью
   - флаг limited (если True, то ограниченная серия)
   - общее количество (тираж)

2. Магазин подарков (UI-компонент):
   - список всех доступных подарков с ценой
   - покупка подарка за звезды (проверка баланса)
   - после покупки открывается окно отправки: выбор получателя (список друзей / контактов) + опциональное сообщение

3. Отправка подарка:
   - создаётся запись в таблице gifts_sent: from_user_id, to_user_id, gift_id, message, status (pending / accepted / declined / hidden), created_at
   - получатель видит уведомление в чате или отдельном разделе «Подарки»

4. Анимация получения:
   - при открытии подарка проигрывается Lottie-анимация (использовать библиотеку lottie-web или lottie-react)
   - после анимации кнопки: «Принять», «Отправить ответный подарок», «Скрыть»

5. Профиль пользователя (новый раздел «Подарки»):
   - отображаются принятые подарки (из таблицы gifts_sent с status = 'accepted')
   - можно закрепить один подарок наверху профиля (поле pinned_gift_id)
   - можно скрыть подарок (status = 'hidden')
   - у ограниченных подарков показывать номер экземпляра (например, #42 / 100)

6. Логика тиражей:
   - при отправке limited-подарка уменьшать остаток в таблице gifts (колонка remaining)
   - нельзя отправить больше, чем осталось
   - если получатель отказывается (declined) — возвращать подарок в тираж (увеличить remaining) и звёзды отправителю
